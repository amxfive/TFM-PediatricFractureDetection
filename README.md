# TFM-PediatricFractureDetection
Diseño, entrenamiento y validación de  un modelo de Deep Learning basado en detección de objetos (Object Detection). El sistema identificará y localizará fracturas en radiografías de miembro superior mediante bounding  boxes (cajas delimitadoras). Se utilizarán imágenes reales proporcionadas por el Hospital  Viamed Santa Ángela de la Cruz.
